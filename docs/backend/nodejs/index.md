# Node.js

Node.js 用于运行 JavaScript 服务端程序，也常用于前端工程化和工具链环境。

常见场景：

- 运行前端项目开发环境
- 使用 `npm`、`pnpm`、`yarn` 管理依赖
- 运行构建工具、脚手架和本地脚本
